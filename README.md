# Abyss Hunter

Abyss Hunter는 모바일 세로 화면을 우선으로 만든 자동 사냥 웹 RPG입니다. 전사 캐릭터를 성장시키고, 장비를 얻고 강화하며, 보스전에 도전하는 방식으로 진행됩니다.

## 실행 방법

별도 설치나 빌드가 필요 없습니다.

1. `index.html`을 브라우저에서 엽니다.
2. GitHub Pages에 올린 경우 Pages URL로 접속합니다.

서버, DB, npm, 빌드 도구를 사용하지 않습니다.

## 기본 플레이 방법

- 사냥 탭에서 자동 전투가 진행됩니다.
- 배속 버튼으로 전투 속도를 바꿀 수 있습니다.
- 장비 탭에서 획득한 장비를 장착, 교체, 해제할 수 있습니다.
- Forge 탭에서 장비 강화, 분해, 자동 분해 설정을 사용할 수 있습니다.
- Boss 탭에서 해금된 보스를 선택해 강타, 방어, 물약, 궁극기를 사용합니다.
- 설정에서 사운드, 진동, 화면 흔들림, 피해 숫자 표시를 조절할 수 있습니다.

## 자동 저장 안내

게임 진행 상황은 브라우저의 `localStorage`에 자동 저장됩니다.

- 같은 브라우저와 같은 주소에서 다시 접속하면 저장 데이터가 유지됩니다.
- 브라우저 데이터 삭제, 시크릿 모드 종료, 다른 주소 접속 시 저장이 사라질 수 있습니다.
- 저장 데이터가 손상된 경우 기본값으로 복구될 수 있습니다.
