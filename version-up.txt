## v0.5

목표:
- 모바일 게임형 고정 화면 UI
- 전투 모션과 타격감 개선
- 효과음, 진동, 화면 흔들림 설정
- 사냥 화면 정보 정리와 하단 시트 활용
- Forge 화면 하위 탭 정리
- 보스전 연출과 스킬 사용감 안정화

상태:
- 완료

현재 반영:
- 고정형 모바일 레이아웃 일부 적용
- 최근 전투 로그 추가
- 상세 능력치와 전체 로그 하단 시트 추가
- Forge 강화, 분해, 자동 분해 하위 탭 분리
- 일반 사냥 공격, 피격, 치명타, 드롭, 레벨업 연출 일부 적용
- 사운드, 진동, 화면 흔들림, 피해 숫자 설정 추가
- v0.5.1 Canvas 전투 화면과 모바일 고정 레이아웃 복구
- v0.5.2 보스전 경고, 방어, 강타, 궁극기 연출과 피드백 강화
- v0.5.3 모바일 최종 안정화, toast 위치 보정, 전투 제목 줄바꿈 정리, 주요 화면 회귀 검증
- v0.5.4 v0.6 준비용 정적 데이터와 CSS 파일 분리

남은 작업:
- v0.6 지역 확장 설계와 구현
